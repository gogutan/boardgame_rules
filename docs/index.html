Board Game Rules

COYOTE
Bluff 2-10 players 15-30 min

A bluffing game where you see everyone else's card but not your own, guess the total, and call Coyote if you think the last number is too high.

ITO
Cooperative 2-10 players 30 min

A cooperative game where each player has a hidden number from 1 to 100 and gives theme-based hints to place the cards in order.
